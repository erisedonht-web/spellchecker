# サモナーズスペルチェッカー 仕様書

## 概要
リーグ・オブ・レジェンドにおいて、ブラウザ上で動作する**敵チームのフラッシュ管理ツール**。

## 前提
- **二画面**：メイン画面（トラッカー）と設定画面を想定。二台モニターでも一画面でも利用可能なUI。
- **対象**：敵の Top / Jungle / Mid / ADC / Support の5ポジションのフラッシュのみ管理。

## 機能要件

### 1. フラッシュ管理
- 各ポジションごとに「フラッシュ使用」をクリックで記録し、経過時間（残りCD）を表示。
- ベースCD：300秒（5分）。短縮オプション適用時は再計算して表示。

### 2. CD短縮のオン・オフ
- **宇宙の叡智（Cosmic Insight）**：18 サモナースペルヘイスト → 各ロールごとにクリックでオン/オフ。
- **アイオニアのブーツ（Ionian Boots of Lucidity）**：10 サモナースペルヘイスト → 各ロールごとにクリックでオン/オフ。
- 適用CD計算式：`実際のCD = 300 / (1 + ヘイスト合計/100)`。各ロールごとの設定でそのロールのCDを計算。

### 3. UI
- ブラウザ上で見やすく、試合中に素早く操作できるレイアウト。
- ダークテーマ想定（LoLクライアントに近い雰囲気）。

## 仕様書の不足点・推奨追加項目

| 項目 | 内容 |
|------|------|
| **他サモナースペル** | 現仕様はフラッシュのみ。イグナイト/テレポート等も管理するか要否の検討。 |
| **味方チーム** | 敵のみと明記済み。味方のフラッシュ管理が必要か。 |
| **リセット** | 各ロールごとのリセットボタンと、全リセットを実装。確認ダイアログなし。 |
| **サウンド** | フラッシュCD終了時の音・通知の要否。 |
| **永続化** | リロードで状態が消える想定か、localStorageで保存するか。→ **実装で保存対応を追加** |
| **数値のパッチ対応** | ゲームのパッチでCDやハステが変わった場合の更新方法（コード内定数で対応）。 |

上記のうち、**リセット**と**localStorageによる永続化**は実装に含めます。他は必要に応じて拡張可能な構造にします。
